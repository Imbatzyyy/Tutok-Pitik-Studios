## 📋 Quick Reference

### Installation
```bash
npm install
cp .env.example .env
npm run dev
```
See: [QUICK_START.md](./QUICK_START.md)

### Build & Deploy
```bash
npm run build
netlify deploy --prod
```
See: [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)

### Environment Variables
```env
VITE_NEON_DATABASE_URL=postgresql://...
VITE_NETLIFY_SITE_URL=https://...
```
See: [ENVIRONMENT_SETUP.md](./ENVIRONMENT_SETUP.md)

### Database Migration
```bash
psql "connection_string" < database/schema.sql
```
See: [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md)
